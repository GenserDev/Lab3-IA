# Lab3-IA

Laboratorio de IA con dos scripts en Python:
- `parte1.py`
- `parte2.py`

## 1) Clonar el repositorio

```bash
git clone https://github.com/GenserDev/Lab3-IA.git
cd Lab3-IA
```

## 2) Crear y activar entorno virtual

```powershell
python -m venv lab03_env
./lab03_env/Scripts/Activate.ps1
```

## 3) Instalar dependencias

```bash
pip install -r requirements.txt
```

## 4) Ejecutar los archivos

```bash
python parte1.py
python parte2.py
```
